docker初步

环境

Ubuntu20.04

docker:24.0.07

安装docker

  1. 需要下载工具：

    • Docker Engine
    • Docker compose

  2. 官方有具体的安装教程

    • Docker Engine ：https://docs.docker.com/engine/install/
    • Docker compose：https://docs.docker.com/compose/install/

  3. 使用snap安装（推荐方法 比较方便）

  4. 注意：使用snap下载之后，之后运行docker需要管理员权限

docker准备工作

  1. 创建一个Docker Hub帐号，链接：https://www.docker.com/products/docker-hub/

  2. 建立一个存储库

  3. 在本地打开一个终端

docker工作基本概念

  1. 镜像：拉取和推送到Docker Hub上面的是在这部分完成
  2. 容器：加载镜像为一个真实可以操作的系统
